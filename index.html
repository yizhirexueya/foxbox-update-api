API 响应格式

API 返回 JSON 格式数据，包含以下字段：

  • code: 状态码，200 表示成功
  • message: 状态描述
  • data: 版本信息对象，包含：
    • version_code: 版本号（整数）
    • version_name: 版本名称
    • update_content: 更新内容
    • download_url: 下载链接
    • force_update: 是否强制更新
    • release_date: 发布日期
